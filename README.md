# awh-clockwork
Final solution for the AWH Clockwork interview exercise.

### How to run the application ###
* Clone this repository
* Open the solution
* Run Clockwork.API
* Update value of "APIUrl" key in Web.config of Clockwork.Web. This is the api url (Clockwork.API).
* Run Clockwork.Web
